说明
==
WeMusic支持：

 - 网易云音乐搜索
 - 酷狗音乐搜索
 - QQ音乐搜索
 - 播放mp3/wav音乐
 - 自动加载封面图片及歌词
 - 可自定义本地歌单
 - 可轻松导入网易云/酷狗/QQ音乐平台歌单
 - 支持单曲循环/随机播放/列表循环
 - 友好、简介的界面设计
